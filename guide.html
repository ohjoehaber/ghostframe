GHOSTFRAME

Repeat-photography field guide · LTPBR stream monitoring

Open it: load the link on your phone's browser, then Share → Add to Home Screen so it launches like an app. The camera, compass and tilt only work over the live web link — a downloaded copy of the file has dead sensors.

1 Start the sensors ONCE / SESSION

Tap Start camera & sensors. Allow camera, location, and — on iPhone — the Motion & Orientation prompt.

  • iOS asks the motion question only once. If someone taps "Don't Allow," the compass and tilt stay blank until they reload the page and allow it.
  • The three pills up top — CAM GPS CMP — turn green when each sensor is live.

2 Build your sites & stations ONCE / POINT

On the Sites tab:

  • New monitoring site → name it, pick structure type (BDA / PALS / BDA+PALS / Reference reach), add the watershed (HUC or name). Add site.
  • Add photo station → choose the site, give a station code (e.g. PS-01), a target heading in degrees (the compass bearing you shoot along), and a placement note (rebar pin, tripod scuff, framing cue). Add station.
  • Each station has a Use button — tap it to make that station active and jump to the camera.

Setup lives on that phone only — set up each phone, or share the station codes & headings so everyone matches.

3 First visit — shoot the BASELINE

  • Pick your site and station in the two dropdowns (the header confirms e.g. PS-01 · tgt 210°).
  • Frame with the rule-of-thirds grid. Rotate until the compass reading (HDG) matches the station's TGT, and keep the level bar green (within ~5° — no sideways tilt).
  • Tap Capture BASELINE. It saves the photo plus timestamp, GPS, heading and tilt.

4 Return visits — line up the REPEAT

  • Same site/station. Tap the Baseline button by the ghost slider — the old baseline fades in over the live view.
  • Slide GHOST to about 50% and physically move until the live scene overlaps the ghost (banks, boulders, the structure).
  • Level green, heading on target → tap Capture REPEAT.

5 Review & hand off the data

On the Gallery tab: filter by site or by baseline/repeat; each card shows station, date, heading, tilt and GPS.

  • Export CSV — metadata table for the spreadsheet.
  • Export JSON — full backup including sites & stations.

⚠ Photos live on the phone only

Nothing is uploaded or synced between devices. Export CSV/JSON before you wipe, upgrade, or risk losing the phone. Treat each phone's data as un-backed-up.

Good habits

  • Reload / pull-to-refresh after an update to get the newest version.
  • Match the station code to your GIS point so the export joins back by code.
  • Shoot with the level green and heading on TGT for clean repeats.
